# bits-frontend

Booking, Inventory, and Tracking System for the Department of Computer, Information System and Mathematics.

## Overview

This is the frontend client for BITS, which was developed using React, TailwindCSS, and Vite. It serves as a role-based user interface for actors, including Admins, LabTechs, LabHeads, Faculty, and Students.

## Tech Stack

- React (Vite)
- TailwindCSS
- Axios
- React Router
- Zustand (for state management, optional)
